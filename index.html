My Projects

Project 1: Smart Home Automation System

This project involves creating a smart home automation system using Arduino and various sensors to control home appliances remotely. The system includes features like:

  • Remote control of lights and fans via a mobile app
  • Temperature and humidity monitoring
  • Motion detection for security
  • Automatic control based on sensor data

The project aims to enhance home convenience and security while promoting energy efficiency.

Project 2: Health Monitoring Wearable

Designed a wearable device to monitor vital signs such as heart rate and temperature, providing real-time data to a mobile app. Key features include:

  • Continuous heart rate monitoring
  • Temperature sensor for body heat detection
  • Bluetooth connectivity for data transmission
  • Mobile app for real-time monitoring and alerts

This project aims to provide users with accessible health data, encouraging proactive health management.

Project 3: IoT-based Weather Station

Developed an Internet of Things (IoT) based weather station to collect and analyze environmental data like temperature, humidity, and pressure. The system features:

  • Accurate environmental data collection
  • Wireless data transmission to a cloud server
  • Data visualization on a web dashboard
  • Alerts for extreme weather conditions

The project aims to provide real-time weather updates, useful for agricultural planning and disaster management.
